# 기능: 인트로 페이지 (첫 진입 화면)

## 목표
비로그인 상태로 루트 주소(`/`)에 접속했을 때 보이는 첫 화면. 서비스의 목적과 핵심 기능을
간단히 설명하고, 카카오 로그인으로 바로 이어지게 한다. 별도 회원가입 폼은 없다
(`docs/features/auth.md` 참고).

## 레이아웃 구성
좌우 2단 구성 (좌: 소개, 우: 로그인 카드) — 참고 레이아웃 구조만 차용, 색상/톤은 반드시
`docs/DESIGN.md` 토큰을 따른다 (참고 이미지의 블루/인디고 계열은 우리 브랜드 컬러가 아님).

### 좌측 — 서비스 소개
- 로고 워드마크 "Paw:Paw" (앞부분 ink, 뒷부분 purple 투톤 — `docs/DESIGN.md` 규칙)
- 슬로건: "우리 아이의 하루를, AI가 함께 돌봐요"
- 한 줄 설명: 반려동물의 건강 기록과 AI 상담을 한 곳에서 관리한다는 요지
- 특징 카드 3개 (숫자 마커 01/02/03 형태로, 실제 우리 핵심 기능을 소개):
  1. **대시보드** — 체중·복약·예방접종을 한눈에
  2. **AI 챗봇** — 증상 상담과 자동 기록
  3. **건강 레포트** — 월간 변화와 AI 소견 요약

### 우측 — 로그인 카드
- Paw:Paw 로고
- "카카오톡 로그인" 버튼 하나만 (이메일/비밀번호 입력 필드 없음 — 참고 이미지와 다른 부분)
- 하단에 짧은 안내 문구 정도만 (예: "카카오 계정으로 간편하게 시작하세요")

## 라우팅
- 비로그인 상태로 `/` 접속 → 이 인트로 페이지 노출
- 카카오 로그인 성공 → 메인페이지(`docs/features/dashboard.md`의 "메인페이지 상세")로 리다이렉트
- 이미 로그인된 상태로 `/` 재접속 → 인트로 페이지를 건너뛰고 바로 메인페이지로 리다이렉트

## 주의사항
- 이 페이지는 로그인 전 유일한 진입점이므로, `docs/DESIGN.md`의 브랜드 톤(민트/퍼플, 아이보리 배경)이 첫인상을 좌우한다 — 참고 이미지의 구조(히어로 텍스트 + 3카드 + 로그인 카드 배치)만 가져오고 색은 절대 그대로 쓰지 않는다
- 회원가입 모달은 만들지 않는다는 기존 규칙과 동일하게, 이 페이지에도 별도 회원가입 폼을 넣지 않는다
