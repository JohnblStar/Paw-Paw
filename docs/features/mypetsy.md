# 기능: 마이펫이지

## 목표
여러 반려동물을 한 계정에서 등록/전환 관리한다.

## 구성
- 반려인 프로필(성함/연령/사진)
- 등록된 반려동물 카드 그리드 — 각 카드에 [부르기] 버튼
- [부르기] 클릭 시 전역 Active Pet ID 변경 → 대시보드/리포트가 해당 동물 데이터로 즉시 갱신
- "새로운 반려동물이 생기셨나요? 추가하기" → 반려동물 추가 화면(기본정보 입력, 세부정보는 이후 챗봇에서)

## API
- `GET /api/v1/users/me/pets` — 등록된 반려동물 목록
- `POST /api/v1/pets` — 반려동물 추가 (기본정보만)
- `PATCH /api/v1/users/me/active-pet` — Active Pet ID 변경

## 주의사항
- 실시간 미리보기 패널 없음 (스코프 아웃 확정)
- "개발자 로그" 버튼은 실사용자 화면에 포함하지 않는다
