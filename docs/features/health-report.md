# 기능: 건강 리포트 (월간 종합 분석)

## 목표
챗봇 상담 및 대시보드 기록을 기반으로 월간 리포트를 생성한다.

## 구성
- 반려동물 기본정보 + 몸무게 기록/추이 그래프 (일/주/월 단위 전환)
- 이번달 최신 증상 + AI 의심사항 (병명 포함 가능 — `chatbot-consult.md`의 `suspectedSymptom` 재사용)
- 이번달 건강 이상 상태 업로드 빈도
- 진단결과 소견: 홈케어 조언(출처 포함) + 병원 방문 시 질문 가이드

## API
- `GET /api/v1/pets/{petId}/report/monthly?month=YYYY-MM`

## 주의사항
- "AI가 분석한 정보는 정확하지 않을 수 있습니다" 같은 면책 문구를 리포트 하단에 항상 노출
- 리포트 생성 자체에 LLM을 추가 호출할지, 챗봇 대화에서 이미 만든 `suspectedSymptom`을 재사용할지는 토큰 예산상 **재사용을 우선**한다 (신규 호출 최소화)
